# Rock Paper Scissors

This is a beginner JavaScript project where the player plays Rock Paper Scissors against the computer.

## What I practiced

- Functions
- Conditionals
- Scope
- Random numbers
- Parameters
- Return values
- Score tracking

## How it works

The player chooses rock, paper, or scissors. The computer randomly chooses one too. The program checks who wins and updates the score.
